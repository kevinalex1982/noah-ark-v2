# 开发约束

> ⚠️ **重要**：每次对话开始时，请让 Claude 读取此文档以恢复关键约束。

---

## 1. 禁止 Claude 启动后端服务

**原因**：飞书 Bridge 是 Node.js 进程，如果 Claude 启动的服务崩溃，可能会导致通话中断。

**禁止的命令**：
- `npm run dev`
- `npm start`
- 任何 MQTT 相关服务启动
- 任何会启动 Node.js 后端进程的命令

**例外**：
- `npm install` ✅ 允许
- `npm run build` ✅ 允许
- `npm run lint` ✅ 允许
- 静态代码检查、依赖安装等不启动长期运行的进程 ✅

---

## 2. 后端服务由用户手动启动

用户会自己启动：
- Next.js 开发服务器
- MQTT Broker
- 其他后端服务

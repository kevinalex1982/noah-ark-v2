## 3. `app/api/credentials/simulate-update/route.ts`

**修改内容：** 添加 userId 提取函数，更新逻辑改为"先删后增"

### 新增函数：extractUserIdFromFeature

```typescript
// 从 featureData 提取 userId（掌纹设备要求 userId 必须和 featureData 里的用户名匹配）
// featureData 结构：[Base64特征数据]=[其他数据][用户名]^^^^...
// 用户名通常是纯小写字母，在 ^ 之前
function extractUserIdFromFeature(featureData: string): string {
  if (!featureData) return '';

  // 找第一个 ^ 的位置
  const firstCaret = featureData.indexOf('^');
  if (firstCaret < 0) return '';

  // 从 ^ 往前找用户名（纯小写字母序列）
  const beforeCaret = featureData.substring(0, firstCaret);

  // 用正则匹配末尾的纯小写字母序列作为用户名
  const match = beforeCaret.match(/([a-z][a-z0-9_-]{2,20})$/);
  if (match) {
    return match[1];
  }

  // 如果没找到，尝试找最后一个 = 后面的内容
  const lastEq = beforeCaret.lastIndexOf('=');
  if (lastEq >= 0) {
    return beforeCaret.substring(lastEq + 1).trim();
  }

  return '';
}
```

## 6. `lib/device-sync.ts` - 虹膜更新流程优化

**修改内容：**
- 导出 `setIrisDeviceSaveState` 函数（锁定/解锁设备）
- 新增 `syncToIrisDeviceWithoutLock` 函数（仅添加，不锁定/解锁）

### 虹膜更新流程

```
删除 → 500ms → 锁定 → 1秒 → 添加 → 500ms → 解锁
```

### 虹膜添加流程（IAMS下发）

```
锁定 → 1秒 → 添加 → 500ms → 解锁
```

**指令间隔：**
- 删除 → 锁定：500ms
- 锁定 → 添加：1秒
- 添加 → 解锁：500ms

```typescript
// === 虹膜更新流程 ===
// 1. 删除
await deleteFromIrisDevice(endpoint, staffNum);
await sleep(500);

// 2. 锁定
await setIrisDeviceSaveState(endpoint, 1);

// 3. 等待1秒
await sleep(1000);

// 4. 添加
await syncToIrisDeviceWithoutLock(endpoint, payload);

// 5. 解锁
await sleep(500);
await setIrisDeviceSaveState(endpoint, 0);

// === 虹膜添加流程（IAMS下发）===
// 1. 锁定
await setIrisDeviceSaveState(endpoint, 1);

// 2. 等待1秒
await sleep(1000);

// 3. 添加
await fetch(endpoint + '/memberSave', ...);

// 4. 解锁
await sleep(500);
await setIrisDeviceSaveState(endpoint, 0);
```

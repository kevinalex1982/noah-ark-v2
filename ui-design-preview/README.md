## 🎨 风格对比

### 风格 1：Glassmorphism 玻璃拟物
- **背景**：蓝紫色渐变 + 浮动光球装饰
- **卡片**：半透明玻璃效果（白色 15% 透明度 + 模糊）
- **按钮**：霓虹光晕效果（青色渐变）
- **适合**：C 端科技产品、需要展现前沿技术形象

### 风格 2：Minimalism 极简
- **背景**：浅灰渐变（非常 subtle）
- **卡片**：纯白 + 细边框
- **按钮**：纯黑色，简洁
- **适合**：企业级应用、金融机构、政府系统
